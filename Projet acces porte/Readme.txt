Introduction:
Ce projet permet aux étudiants d'accéder à la serre sans avoir besoin de demander une clef à un professeur. Ceci permet d’utiliser le moins de ressources humaines possible, car l’étudiant peut par lui-même faire une demande d’accès en quelques secondes et par la suite recevoir le code du cadenas. Le projet ajoute une couche de protection parce que la demande est envoyée à un courriel que seulement les étudiants ont accès, donc il est impossible d'entrer dans la serre sans être un étudiant. 

Fonctionnement du projet:
Pour utiliser le projet de contrôle d’accès, il faut tout d’abord être un étudiant du collège de Maisonneuve. Une feuille d’instruction est présente à la serre pour guider les étudiants afin d'accéder à celle-ci. En ce qui concerne les instructions, voici les étapes à suivre: 

1.	L’utilisateur doit entrer son numéro d’étudiant à 7 chiffres. 
2.	Il doit attendre un instant pour que le programme lui envoi le code du cadenas a son courriel d’étudiant.
3.	À l’aide de son téléphone cellulaire, l’étudiant doit se connecter à son courriel de l’école (Ex: e1234567@cmaisonneuve.qc.ca).
4.	Dans le courriel, l’étudiant aura accès au code du cadenas et par la suite il aura accès à la serre.

***Si une erreur a été commis durant l’étape 1, vous devez appuyer sur la touche ‘#’ pour effacer et recommencer à nouveau.
